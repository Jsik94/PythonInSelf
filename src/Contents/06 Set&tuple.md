## set
***
중괄호로 표현하며, 순서가 없다.(단,숫자로만 이루어진 집합은 오름차순 출력)
중복이 없다. (같은건 하나만 가능)
update()를 통해서 추가가 가능하고, remove()를 통해서 삭제가 가능

```python
print(set(i for i in 'PYTHON LOVE'))# 중복인 [o]는 한번만 집합에 들어감
```
